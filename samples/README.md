# Sample Index

This folder contains all project audio assets used by the SuperCollider setup.

## Folder Structure

- `samples/breaks/` - core breakbeats
- `samples/bass/` - bass and sub layers
- `samples/atmos/` - pads, vocals, long ambient and melodic layers
- `samples/fx/` - one-shot and transitional FX
