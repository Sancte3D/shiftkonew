SHIFTKO CURRENT FUNCTIONS / CONTROLS (SuperCollider Monolith)

Core architecture:
- Single monolithic live instrument with:
  - Ambient engine (drone/notes/texture/noise)
  - DnB slicer subsystem
  - Vocal granular subsystem
  - MIDI learn/mapping and feedback
  - Session save/load

Sample libraries in code:
- ~samplePaths[\breaks]
- ~samplePaths[\bass]
- ~samplePaths[\fxCore]

==============================
UI: IMPORTANT CURRENT LOGIC
==============================

Basic UI mode:
- Button: BASIC UI ON / BASIC UI OFF
- Default is BASIC UI ON.
- Basic mode shows only core controls + session + DnB/Vocal blocks.

Session controls:
- SAVE SESSION
- LOAD SESSION
- RESET SESSION

Safety / behavior controls:
- STRICT PICKUP ON/OFF
- PICKUP OVERRIDE ON/OFF
- PERF SAFE LOCK ON/OFF
- AUTO GUARD ON/OFF

DnB simplified controls:
- DNB OFF/ON (single toggle flow)
  - ON auto-loads random break if no break loaded, then starts.
  - OFF stops dnb player.
- NEXT BREAK (loads random break, restarts if currently running)
- Knobs:
  - DnB Vol
  - Chaos
  - Roll
  - Stutter
  - Rate Off
  - DnB Rev
  - Tgt BPM
  - Src BPM
  - Break HPF
  - Kick Amp
  - Sidechain

Vocal granular simplified controls:
- VOC OFF/ON (single toggle flow)
  - ON auto-loads random vocal if none loaded, then starts granular synth.
  - OFF stops granular synth.
- NEXT VOCAL (loads random vocal, restarts if currently running)
- Display: current vocal file name
- Knobs:
  - V Amp
  - V Rate
  - V Pos
  - V Jitter
  - V Dur
  - V Dens
  - V HPF
  - V LPF
  - V Rev
  - V Pan

Arrangement block:
- ARRANGE OFF/ON (task-based intro/drop/breakdown loop)
- Durations:
  - I sec
  - D sec
  - B sec
- Phase editor:
  - Phase selector (intro/drop/breakdown)
  - Param selector
  - Value field
  - APPLY ARR (single param write)
  - APPLY PHASE NOW (batch apply full phase)
- Macro slots:
  - SAVE/LOAD ARR A
  - SAVE/LOAD ARR B
  - SAVE/LOAD ARR C
